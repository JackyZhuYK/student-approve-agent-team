# student-approve-agent-team

AI Mock 学生审批 Agent 团队项目

## 项目简介

此项目用于学生审批相关的 Agent 团队协作功能。

## 快速开始

```bash
# 安装依赖
npm install

# 启动项目
npm run dev
```
